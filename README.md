# VaaniAI+ – Voice Intelligence System

## 🚀 Overview
VaaniAI+ is a real-time voice assistant that enables natural interaction using AI and Murf Text-to-Speech.

## ✨ Features
- Voice-based interaction
- Context-aware responses
- Real-time processing
- Scalable architecture

## 🛠 Tech Stack
- Node.js
- Express
- Web Speech API
- Murf AI (planned integration)

## 🎯 Use Cases
- Education
- Productivity
- Voice assistants
